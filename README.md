# Quote of the Day

A simple JavaScript practice project that displays random quotes in both English and Arabic.

## Features
- Random quote generation using JavaScript
- Support for English (LTR) and Arabic (RTL) content
- Clean separation between HTML, CSS, and JavaScript
- Simple and consistent UI styling

## Purpose
This project was built as a small practice task to improve:
- JavaScript logic
- DOM manipulation
- Handling text direction (RTL / LTR)
- UI consistency and clean structure

## Technologies Used
- HTML
- CSS
- JavaScript
- Bootstrap
